# 🌊 FoodieFlow - Premium React Food Delivery Ecosystem

A futuristic, high-end food delivery platform featuring a **Liquid Glass (Water UI)** design system. Built with **React.js** and **MERN** stack for a professional internship-level project.

## ✨ Premium Features

- **🛡️ Multi-Role Hub**: Seamless transitions between Customer, Restaurant, and Delivery agent portals.
- **💧 Water UI Design**: Deep navy gradients, electric cyan glows, and liquid glass frosting effects.
- **🚀 Warp Speed UX**: Framer-motion animations and micro-interactions for a premium SaaS feel.
- **🛰️ Live Tracking**: Real-time order status and agent location tracking via Socket.io.
- **📊 Metric Ripples**: Professional analytics dashboard for restaurant owners.
- **🗺️ Map Flow**: High-precision Google Maps integration for address selection and routing.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite, Framer Motion, Lucide Icons, Axios.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Real-time**: Socket.io.
- **Auth**: JWT (JSON Web Tokens) with Secure LocalStorage.

## 🚀 Rapid Setup

### 1. Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)

### 2. Backend Setup
```bash
cd server
npm install
node server.js
```

### 3. Frontend Setup
```bash
cd client
npm install
npm run dev
```

Visit `http://localhost:5173` to experience the flow.
